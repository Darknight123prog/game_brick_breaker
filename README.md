# 🎮 Brick Breaker Game

A simple **Brick Breaker** game built using **HTML5 Canvas, CSS, and JavaScript**.  
Control the paddle, bounce the ball, and destroy all the bricks to win!  

---

## ✨ Features
- Paddle movement controlled with keyboard
- Bricks arranged in rows and columns
- Score tracking
- Game over and restart option
- Background image support
- Fully responsive canvas

---

## 🚀 How to Play
1. Open the game in your browser.
2. Press the **Start** button.
3. Use:
   - **Left Arrow (←)** to move left  
   - **Right Arrow (→)** to move right  
4. Keep the ball from falling and break all the bricks!
